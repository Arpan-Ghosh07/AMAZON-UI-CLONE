# 🛒 Amazon UI Clone

A visually accurate **Amazon homepage UI clone** built using modern front-end technologies.
This project focuses on replicating the look and feel of Amazon’s interface while strengthening core frontend development skills.

---

## 📌 Features

* 🧭 **Navigation Bar**

  * Amazon logo
  * Search bar
  * Location selector
  * Sign-in & cart options

* 🎞️ **Hero Section / Banner**

  * Promotional banner similar to Amazon homepage

* 🛍️ **Product Sections**

  * Multiple product cards with images and titles
  * Category-based layout

* 📦 **Footer**

  * Multi-column footer similar to Amazon
  * Organized links and sections

---

## 🛠️ Tech Stack

* **HTML5** – Page structure
* **CSS3** – Styling (Flexbox & Grid)
* **JavaScript** – Basic interactivity (if implemented)

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/Arpan-Ghosh07/AMAZON-UI-CLONE.git
```

2. Navigate to the project folder

```bash
cd AMAZON-UI-CLONE
```

3. Open in browser

```bash
open index.html
```

(or simply double-click the file)
